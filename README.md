# Student Job & Internship Tracker

A simple web application to help students track their job and internship applications. 
Users can add applications, set their status, delete entries, and data is saved using browser LocalStorage.

## Features
- Add job/internship applications with company, position, and status
- Delete applications
- Status options: Applied, Interview, Offer, Rejected
- Data is saved in browser LocalStorage (persistent)
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## How to Use
1. Download or clone the repository
2. Open `index.html` in your browser
3. Add your job or internship applications using the form
4. View them in the table and manage status or delete entries
